# 题目描述
某国为了防御敌国的导弹袭击，发展出一种导弹拦截系统。但是这种导弹拦截系统有一个缺陷：虽然它的第一发炮弹能够到达任意的高度，但是以后每一发炮弹都不能高于前一发的高度。某天，雷达捕捉到敌国的导弹来袭。由于该系统还在试用阶段，所以只有一套系统，因此有可能不能拦截所有的导弹。

输入导弹依次飞来的高度，计算这套系统最多能拦截多少导弹，如果要拦截所有导弹最少要配备多少套这种导弹拦截系统。

### 输入格式
一行，若干个整数，中间由空格隔开。

### 输出格式
两行，每行一个整数，第一个数字表示这套系统最多能拦截多少导弹，第二个数字表示如果要拦截所有导弹最少要配备多少套这种导弹拦截系统。

### 数据范围
对于前50%数据，满足导弹的个数不超过10<sup>4</sup>个。对于后50%的数据，满足导弹的个数不超过10<sup>5</sup>个。
对于全部数据，满足导弹的高度为正整数，且不超过 5×10<sup>4</sup>。

# 问题分析
### 问题转化
对于第一个问题，显然只需求出最长的非严格递减子序列的长度即可；而对第二个问题，为了求出需要配备多少种导弹拦截系统，根据Dilworth定理，链的个数等于最长严格递增子序列的长度。因此可以将整个问题视为一个纯粹的问题：求最长递增/递减子序列长度
### 原始思路
以求最长（非严格）递减子序列长度为例。要想知道最长的不上升的子序列长度，只需找到所有序列长度的最大值，由此产生朴素的思路：遍历所有高度值，每当遍历一个新的高度时，先查询在它之前且高度不小于它的所有子序列长度，取其最大值加一，即为当前的最长递减子序列长度
### 方法优化
不难看出，在最原始的思路中，需要遍历所有高度值，然后对每个高度值，都要遍历此前的高度以取最大值，时间复杂度为O(n<sup>2</sup>)，不能满足题目的要求
如果要对时间复杂度进行优化，在不跳出现有框架的情况下，以遍历高度为优化目标是比较困难的，但取高度最大值的过程则有很大的优化空间。我们永远只需要知道前若干项的最大值，因此也可以使用类似快速求前若干项之和的方法：最大值树状数组。
# 实现思路
#### Step1 离散化
为了便于使用最大值树状数组，首先对输入的高度序列进行离散化，将高度值改为高度排名。
##### 例子
输入为：389 207 155 300 299 170 158 65，离散化后变成：1 4 7 2 3 5 6 8
#### Step2 遍历数组
遍历离散化之后的数组，对每个值，都找到之前遍历过的、高度不小于自己的值所记录的最大长度，将其加一作为自己这一项的最大长度
##### 例子
遍历到1，第一位之前没有数据，因此最大值取0，此处最大长度为0 + 1 = 1，在数组下标为1处记录长度为1；然后遍历到下一项为4，下标小于4的项中，只有1刚才被赋值为1，其余均为默认值0，故高度不小于自己的项中，最大长度为1，此处的最大长度为1 + 1 = 2，在数组下标为4处记录长度为2.依此类推，直到遍历完整个数组。
需要注意的是，以上所说的“记录长度为x”仅仅是便于理解，实际记录的时候，需要按照树状数组的规则，在更新值之后向上传播，对数组内相关的项进行更新。
#### Step3 得出结果
由于已经得到了以每一项作为结尾的最大递减序列的长度，只需找出这8项的最大值即可，而这一操作在最大值树状数组中可以快速高效地完成

# 最终代码

```cpp
#include <bits/stdc++.h>
using namespace std;

class Analyzer{
private:
    vector<int> increase;  // 树状数组，用于维护以每个值结尾的最长严格递增子序列长度
    vector<int> decrease;  // 树状数组，用于维护以每个值开头的最长非严格递减子序列长度（通过值域反转实现）

    // 查询树状数组中 [1, idx] 范围内的最大值
    // pattern == true 表示查询 increase 数组，否则查询 decrease 数组
    int getMax(int idx, bool pattern){
        int ans = 0;
        vector<int>& tree = pattern? increase: decrease;
        for(int i = idx; i > 0; i -= (i & -i)) ans = max(ans, tree[i]);
        return ans;
    }

    // 在树状数组的 idx 位置更新值
    // pattern == true 表示更新 increase（递增子序列），否则更新 decrease（递减子序列）
    void add(int idx, bool pattern){
        // 对于递增：查询 [1, idx-1] 的最大 LIS 长度；对于递减：查询 [1, idx] 的最大 LDS 长度（因值已反转）
        int q = pattern? (idx > 1? getMax(idx - 1, true): 0): getMax(idx, false);
        int val = q + 1;  // 当前位置可形成的子序列长度
        vector<int>& tree = pattern? increase: decrease;
        // 树状数组单点更新：将 val 向上传播，维护区间最大值
        for(int i = idx; i <= (int)tree.size() - 1; i += (i & -i)){
            tree[i] = max(tree[i], val);
        }
    }

public:
    // 构造函数：对输入数组 org 预处理，构建 increase 和 decrease 树状数组
    Analyzer(const vector<int>& org): increase(org.size() + 1, 0), decrease(org.size() + 1, 0){
        int n = org.size();
        vector<int> sorted = org;
        sort(sorted.begin(), sorted.end());  // 离散化用

        for(int i = 0; i < n; ++i){
            // 获取 org[i] 在排序后数组中的离散化位置（从1开始）
            int position = lower_bound(sorted.begin(), sorted.end(), org[i]) - sorted.begin() + 1;
            add(position, true);  // 更新递增子序列信息
            // 将值域反转：大值变小，小值变大，从而用相同结构处理递减子序列
            add((int)sorted.size() - position + 1, false);
        }
    }

    // 获取最长子序列长度：pattern == true 为 LIS，false 为 LDS
    int getLength(bool pattern){
        return getMax((int)increase.size() - 1, pattern);
    }
};

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string line;
    getline(cin, line);
    stringstream ss(line);
    vector<int> nums;
    int x;
    while(ss >> x) nums.push_back(x);

    Analyzer analyzer(nums);
    cout << analyzer.getLength(false) << endl << analyzer.getLength(true) << endl;
    return 0;
}
```
